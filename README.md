
## SPA deploy

> pm2-deploy-react-project

## Build Setup

``` bash

# git clone
git clone https://github.com/oksite/pm2-deploy-react-project.git

# pm2 install dependencies
npm install -g pm2 

# install dependencies
npm insatll

# serve with hot reload at localhost:3000
pm2 start 

# close pm2 server 
pm2 stop 

# deploy server 
pm2 deploy 

```
